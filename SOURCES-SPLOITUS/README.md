# CVER : SOURCES-SPLOITUS.

## JSON : code.

    [
      {
        "source": "*",
        "id": "*",
        "href": "*",
        "title": "*"
      }
    ]

## JSON : detail.

- source : The source code of the sploitus "ID".
- id : The "ID" of the sploitus.
- href : The "Github repos" related to the sploitus "ID".
- title : The title of related sploitus "ID".
